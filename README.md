Difference between var, let, and const: all three words are used to declare variables. But var is now obsolete because its value can be changed before its creation and value assignment. When the value can be changed we usually use let and where the value is fixed const is used.
Spread operator is used to spread out elements of an array or an object.
forEach does not return a new array, map() returns a completely new array but does not change the original one, and filter() also creates a new array with the element which is passed by the function.
when a function is written using =>, then it is an arrow function.
template literals are written like this: ${}, it is used to change an element dynamically.
